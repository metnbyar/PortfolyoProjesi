




# **.NET MVC ile Geliştirilmiş Portfolyo Sitesi**


## **Bu projede kullanılanlar**


* Model-View-Controller katmanları

* Layout Kontrolü

* Admin Paneli

* Code First

* EntityFramework

* Authentication ve Authorization 

* CRUD işlemleri

* HttpGet ve HttpPost attributeleri

* Validation kontrolleri
